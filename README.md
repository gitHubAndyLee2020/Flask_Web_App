# Flask Web App Project

This is a full-stack web app built with Flask, where the user can signup, login, logout, and create a simple list of notes.

Tech Stack:

- Flask
- Jinga
- flask-login
- flask-sqlalchemy

---

Installations:

```bash
pip install flask flask-login flask-sqlalchemy
```

---

Running The App:

```bash
python main.py
```
